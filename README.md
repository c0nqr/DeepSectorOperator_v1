# Deep Sector Operator

A space exploration and resource management game set in a vast, mysterious sector of space.

## Game Overview

Deep Sector Operator is an experimental space game that combines elements of:
- Resource gathering and management
- Combat and defense mechanics
- Exploration and navigation through space sectors
- Strategic decision-making about when to retreat or press forward

## Core Gameplay Features

### Sector Navigation
- Navigate through interconnected space sectors
- Jump between sectors using gate systems
- Manage fuel consumption during travel

### Resource Management
- Mine resources from nodes in each sector
- Collect various materials and components
- Balance resource collection with other activities

### Combat System
- Defend against enemy attacks
- Engage in tactical combat scenarios
- Protect your freighter from threats

### Boss Battles
- Face challenging boss enemies
- Learn their attack patterns and weaknesses
- Plan your strategy for victory

### Freight Transport
- Ship resources back to base
- Manage cargo capacity and delivery routes
- Coordinate with freighter systems

## Controls

### Movement
- **WASD** or **Arrow Keys**: Move your ship
- **Mouse**: Aim and target enemies

### Actions
- **Spacebar**: Fire weapon
- **M**: Call freighter for transport
- **Q**: Cycle targets

## Project Status

This is an ongoing development project. The current build includes core gameplay mechanics but is still in active development.

## Development Notes

The project is built using Godot 4.x with GDScript. The architecture follows a modular approach with:
- Autoloads for global game state management
- Scene-based organization for different game elements
- Input system configuration for player controls
